JavaServer Faces (JSF) is a MVC web framework that simplifies the construction of User Interfaces (UI) 
for server-based applications using reusable UI components in a page. 

JSF provides a facility to connect UI widgets with data sources and to server-side event handlers. 

The JSF specification defines a set of standard UI components and provides an Application Programming Interface (API) 
for developing components. JSF enables the reuse and extension of the existing standard UI components.
